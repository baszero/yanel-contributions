About

We want to make social networks open. Open networks mean communication without boundaries and lock-in.

The strategy is very simple: We will provide open social network server and client software for everyone such that everyone is empowered to become part of a decentralized network (just as email is today).

In order to guarantee openness we have defined "The three laws of an open social network provider":

  1. An open social network provider must provide an export/migrate button, which allows one to move his/her own data/profile completely to any other "open social network provider"
  2. An open social network provider must provide a delete button, which allows one to delete his/her own data/profile completely
  3. An open social network provider must protect one's data/profile, as long as such protection does not conflict with the First or Second Law.

Please find more information about the technical implementation at http://yanel.wyona.org/specification/foaf/index.html.

The FOAF Yanel Realm was originally created by Michael Wechner.
