Importing FOAF files

  • Add the following line to your $YANEL_HOME/conf/local/local.resource-types.xml 
    <resource-type src="/home/USERNAME/wyona/public/yanel/contributions/realms/foaf/src/resources/import-foaf/" compile="true"/>
  • Re-build and re-start Yanel.
  • Update access rights by editing FOAF_REALM_SRC/ac-policies/import-foaf-files.html.policy
  • Browse to http://127.0.0.1:8080/yanel/foaf/import-foaf-files.html
  • Change the Import directory within FOAF_REALM_SRC/res-configs/import-foaf-files.html.yanel-rc to the directory where the FOAF files you would like to import are stored.
  • Reload page http://127.0.0.1:8080/yanel/foaf/import-foaf-files.html and click 'Import'.
  • Search for some keyword to test the import.
